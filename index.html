Logo Usage Requirements
  • Always maintain clear space equal to the height of the dot around the logo
  • Never stretch, distort, rotate, or apply effects to the logo
  • Minimum size: 32px height for digital, 0.5 inches for print
  • Use appropriate version based on background color and context
  • Never place logo on busy backgrounds without sufficient contrast

Typography Guidelines
  • Display & Headings: Use Satoshi (weights 600-900) for all display text and headings
  • Body & UI: Use Inter (weights 400-700) for paragraphs, buttons, and interface elements
  • Minimum Sizes: 14px for body text, 12px for captions and labels
  • Line Height: 1.5-1.7 for body text, 1.1-1.3 for headings
  • Type Scale: Follow the Perfect Fourth scale (1.333 ratio) for consistent hierarchy
  • Both fonts are available free via Google Fonts and Fontshare

Usage Guidelines

Best practices and requirements for maintaining brand consistency across all applications.

Logo Usage

Do
  • Use the primary Electric Cyan logo on light backgrounds
  • Maintain proper clear space equal to the dot height around logo
  • Use the reversed white version on dark backgrounds or photography
  • Scale proportionally maintaining aspect ratio
  • Use approved color variations for appropriate contexts
  • Ensure minimum size requirements are met for legibility
Don't
  • Never stretch, distort, or alter the logo proportions
  • Don't rotate, skew, or apply perspective transformations
  • Never use unapproved colors or create new variations
  • Don't place on busy backgrounds without proper contrast
  • Never add effects like drop shadows, glows, or gradients to the logo
  • Don't recreate or redraw the logo - always use official files

Color Usage

Do
  • Use Electric Cyan for primary CTAs and brand elements
  • Use Coral Red for secondary actions and accents
  • Ensure all text meets WCAG contrast requirements (4.5:1 minimum)
  • Use gradients sparingly for backgrounds and decorative elements
  • Maintain color consistency across all brand touchpoints
  • Test colors in both light and dark environments
Don't
  • Never apply gradients directly to the logo itself
  • Don't use more than 3 colors in a single composition
  • Avoid using pure black (#000000) - use Deep Navy instead
  • Don't create new color variations without approval
  • Never compromise accessibility for aesthetic choices
  • Don't use colors at opacities that fail contrast requirements
Core Brand Principles
  • Modern & Minimal: Embrace white space and clean layouts
  • Bold & Vibrant: Use color confidently to create impact
  • User-Focused: Prioritize clarity and usability in all applications
  • Consistent: Apply guidelines uniformly across all media and platforms
  • Accessible: Ensure content is perceivable and usable by everyone
